# Software Engineer — Backend (Python)

## About the Role
We are looking for a backend engineer to join our platform team.
You will design and ship RESTful APIs, own microservices, and contribute to 
our MLOps infrastructure.

## Requirements (Must Have)
- 2+ years of professional Python development
- Experience with FastAPI or Django
- SQL databases (PostgreSQL preferred)
- Git-based workflows and CI/CD pipelines
- Experience writing unit and integration tests

## Requirements (Nice to Have)
- Docker and Kubernetes
- Experience with machine learning pipelines or MLOps tools
- Redis or message queue experience (Kafka, RabbitMQ)

## Responsibilities
- Build and maintain production Python services
- Collaborate with the ML team to productionize models
- Participate in code review and technical design
- On-call rotation (one week per quarter)

## Team
Small team (8 engineers), fast-paced, Python-first stack, remote-friendly.
